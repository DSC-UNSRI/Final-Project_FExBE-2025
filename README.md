## ğŸ“– Cara Menggunakan Repository Ini

### Clone Repository Beserta Submodules

```bash
# Clone main repository
git clone https://github.com/DSC-UNSRI/Final-Project_FExBE-2025.git

# Masuk ke folder project
cd Final-Project_FExBE-2025

# Initialize dan download semua submodules
```
